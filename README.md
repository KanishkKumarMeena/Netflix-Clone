
# Netflix Frontend Clone <br/>

I developed the frontend of a Netflix clone, an online streaming platform with an intuitive and visually appealing user interface. The design ensures ease of use, allowing users to effortlessly browse and watch their favorite shows and movies. The frontend features dynamic content loading, smooth transitions, and a responsive layout, providing a seamless viewing experience across various devices.<br/>
